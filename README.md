# VIPERMovies
 
<b> A movie search app that using the VIPER architecture.</b>
  
 ## <b> Features, Libraries </b>  
   - VIPER
   - Alamofire
   - Kingfisher
   - Lottie
   - No Storyboard, Programmatic UI
   - AutoLayout
